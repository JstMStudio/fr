Discoserveur

Discoserver est une bibliothèque gratuite de serveurs de discussion sur plusieurs plates-formes. Discoserver a des serveurs sur Discord, Guilded et Revolt sur notre site Web pour que tout le monde puisse voir gratuitement tous les serveurs et bots de Discoserver. Nous avons également des listes de bots Guilded, Discord et Revolt développés par les communautés. Nous avons également une liste de thèmes dans Revolt aussi ! Les utilisateurs peuvent également ajouter un nombre illimité de serveurs/bots/thèmes dans la bibliothèque Discoserver gratuitement et sans frais.


Discoserver est un logiciel open source (OSS) ! Le code source de Discoserver est publiquement accessible à tous sous licences MIT. Tout le monde pouvait voir, étudier et lire tout notre code source à tout moment, n'importe quel jour de l'année.
