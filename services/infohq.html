Couleur

Les utilisateurs peuvent également modifier la couleur du texte dans tous les InfoHQ. Vous pouvez modifier toutes les couleurs de texte dans les pages et les sites Web d'InfoHQ. Par défaut, la couleur du texte InfoHQ est verte avec le pied de page bleu. Vous pouvez le changer en n'importe quelle couleur que vous aimez si vous le souhaitez.

Modifier

Toutes les pages d'InfoHQ sont modifiables par tout le monde, de l'administrateur aux utilisateurs normaux et aux membres. Les utilisateurs peuvent modifier toutes les pages et sites Web d'InfoHQ, comme aider à ajouter plus d'informations, etc. Vous pouvez modifier toutes les pages de votre InfoHQ en visitant le chemin : infohq /(nom de la base de données InfoHQ)/ dans le répertoire principal de notre site Web. Une fois que vous avez terminé, vous pouvez soumettre une demande d'extraction pour attendre qu'un examinateur de demande d'extraction l'approuve afin qu'elle puisse être mise en ligne sur les sites Web d'InfoHQ.

    Installation

    Afin d'installer InfoHQ, vous aviez d'abord besoin d'un hébergement Web prenant en charge l'hébergement de sites statiques. La plupart des hébergements Web gratuits et payants prennent également en charge les sites statiques. Vous pouvez télécharger InfoHQ dans l'onglet de publication du référentiel Github. Vous pouvez également télécharger n'importe quelle version que vous aimez ici. Après le téléchargement, veuillez télécharger dans votre section de fichier d'hébergement. Une fois que vous avez téléchargé et créé le fichier, modifiez les liens "Modifier", "Afficher la source" et "Historique" et appuyez à nouveau dessus. Une fois le push réussi, votre InfoHQ est maintenant opérationnel pour que vous puissiez créer et modifier des articles.
